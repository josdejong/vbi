# VBI

An interactive business intelligence tool

## Usage

```bash
npm install
```

Start the development server with hot reloading:

```bash
npm start
```

Or start the server at a custom port:

```bash
PORT=5000 npm start
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
